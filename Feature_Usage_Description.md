# Feature Usage Description / 功能敘述使用說明

本文件說明「智慧穿搭顧問平台」的詳細功能操作指南，包含現有功能與即將推出的個人化/進階推薦功能。

---

## 1. 智慧衣櫥 (Smart Wardrobe)

### 1.1 上傳新衣物
1.  點擊首頁或導覽列的 **「上傳 (Upload)」** 按鈕。
2.  選擇相簿中的衣物照片（支援多張選取）。
3.  系統將自動進行 **AI 辨識**：
    *   分析衣物的 **顏色**。
    *   分類為 **上衣/下身/外套/連身裝/配件**。
    *   歸類為 **15 種標準風格** 之一 (如：日系、韓系、美式復古...)。
4.  上傳完成後，衣物會自動顯示在「我的衣櫥」中。

### 1.2 管理衣櫥
*   **瀏覽**: 在首頁下方可瀏覽所有已登錄的衣物。
*   **編輯/刪除**: 點擊單品卡片，可手動修正 AI 判斷錯誤的標籤，或刪除不再使用的衣物。

## 2. 個人設定 (Personal Settings) *[New]*
*此頁面用於建立您的數位時尚檔案，讓 AI 更懂您的需求。*

### 2.1 建立基本資料
1.  進入 **「個人設定 (Profile)」** 頁面。
2.  輸入您的 **身高 (cm)** 與 **體重 (kg)**：這將協助 AI 提供修飾身形的建議。
3.  設定 **體感偏好 (Thermal Preference)**：
    *   `怕冷 (Cold Sensitive)`：AI 會建議比標準天氣更保暖的層次。
    *   `正常 (Normal)`：依照氣溫標準推薦。
    *   `怕熱 (Heat Sensitive)`：AI 會優先推薦透氣、輕薄的組合。

### 2.2 設定風格 DNA
1.  **風格偏好**: 使用下拉選單選擇您喜歡的風格（例如 "Japanese Cityboy"）。
    *   系統會即時顯示該風格的詳細定義（如：寬鬆、大地色系...）。
    *   點擊「新增」將其加入您的喜好清單（可複選）。
2.  **避雷清單 (Dislikes)**: 輸入您絕對不想穿的單品關鍵字（例如："短褲", "動物紋", "高跟鞋"）。AI 推薦時將嚴格避開。
3.  **自訂描述**: 可自由輸入更多文字備註給 AI 參考。

### 2.3 查看歷史紀錄
*   在個人設定頁面下方，可查看過去每一次 AI 幫您搭配的紀錄、日期與場合。

## 3. 智慧穿搭推薦 (AI Outfit Recommendation)

### 3.1 基礎推薦流程
1.  在首頁的 **「場合 (Occasion)」** 輸入框，輸入您今天的目的地（例如：「週末去海邊咖啡廳」、「週一公司重要會議」）。
2.  在 **「風格 (Style)」** 輸入框，輸入當下想要的感覺（或留白以使用個人預設值）。
3.  系統會自動抓取 **當地天氣**，並生成 3 套適合的穿搭建議。
4.  每套建議包含：
    *   **搭配單品**: 從您衣櫥挑選的具體衣服。
    *   **穿搭顧問建議**: 針對您身形與當天天氣的 100 字短評。

### 3.2 指定單品搭配 (Anchor Item) *[New]*
*當您只想穿某件衣服，卻不知如何搭配時使用。*
1.  在推薦介面點擊 **「指定單品」**。
2.  從衣櫥清單中勾選這一件（或多件）衣服（例如：一件新買的紅色裙子）。
3.  按下生成，AI 將圍繞著這些鎖定的單品，幫您補完剩下的全身穿搭。

### 3.3 缺件導購 (Shop the Look) *[New]*
1.  若 AI 認為某個風格極度適合您，但您衣櫥中缺乏關鍵單品（例如：缺一件「米色風衣」）。
2.  推薦結果卡片中會出現 **「推薦入手」** 區塊。
3.  點擊連結可直接搜尋外部電商平台，幫助您補齊衣櫥缺口。
